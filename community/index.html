Contribuindo com o Projeto

Mono é um projeto open source o código está disponível no GitHub. Para contribuir com bug fixes ou novas features, apenas fo fork o repositório e envie para nós um pull request!

Veja o guia de como contribuir para mais informações e uma introdução no processo de desenvolvimento.

Relato de Bugs

Quando você encontrar algo que não funcione como deveria,nós adorariamos se você nos enviase um relato sobre o bug no Xamarin's Bugzilla.

Tenha certeza de incluir detalhes na descrição do problema e os passos necessários para reprosuzi-lo. Anexe Attaching um pequeno test case isso tornará muito mais simples para nós.

Veja instruções detalhadas de como fazer um bom relato de bug em nossoguia bug reporting.

Conseguindo Ajuda

Nós temos muitas lista de email aqui você pode perguntar por ajudaem qualquer assunto relacionado ao Mono. Estas listas estão disponíveis por simples interface web também.

Você pode participar da nossa lista brasileira em monobrasil@googlegroups.com e participar desssa comunidade, ou na nossa Fan Page , ou do nosso grupo no FaceBook

Outra forma de conseguir ajuda é no chat IRC . Veja em IRC para maires informações sobre como conectasr ao nosso servidor IRC.

Stack Overflow é be, popular de perguntas e resposta; Neste site os usuários podem fazer perguntas sobre desenvolvimento de software, incluindo Mono. Muitos contribuidores do Mono e membros ativos da comunidade vão respoder perguntas marcadas com a Tag MONO no Stack Overflow, tenha certeza que suas questões tem a tag mono.
