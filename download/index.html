Mono for Mac OS X is available as a Mac Package (.pkg)

Please refer to the installation guide for more information about how to install and configure your Mono environment.

Download Mono MRE Download Mono MDK

MRE is the Mono Runtime Environment, MDK is the Mono Development Kit.

Use the MDK if you're developing applications that run on .NET/Mono. It includes the MRE and extra tools, libraries, .NET PCL (Portable Class Library) profiles, etc. that are required for development.

Mono for Linux is available for various distributions

Please refer to the installation guide for more information about how to install and configure your Mono environment.

Xamarin packages

These packages are built, tested and distributed by Xamarin. Use these if you want to use a stable, official version of Mono.

Mono packages for Debian, Ubuntu, and derivatives

While the APT package is built against Debian Wheezy, it is compatible with a number of Debian derivatives, including Ubuntu 14.04 LTS. See the installation guide for detailed installation instructions across many versions of Debian and Ubuntu.

Apt repo: deb http://download.mono-project.com/repo/debian wheezy main

Mono packages for Red Hat, SUSE, and derivatives

Yum repo: http://download.mono-project.com/repo/centos/

Linux distribution packages

Mono is packaged in various Linux distributions by third-party maintainers. Some packages do not ship the latest Mono version; this depends on the distribution/maintainer.

  • Arch Linux
  • CentOS
  • Debian
  • Gentoo
  • openSUSE
  • Ubuntu

Continuous Integration packages

Every commit to our repository automatically builds distribution packages. Use these if you want a bleeding edge Mono version and don't care about stability.

Learn more about CI packages

Community packages

These packages are provided by the community and linked here for your convenience. They are not tested by Xamarin, so use them at your own risk.

  • Mono Packages installed to /opt for CentOS, Fedora, Debian and Ubuntu
  • Latest SUSE mono packages, part of the official "Factory" repository
