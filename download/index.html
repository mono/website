O MDK package é para desenvolvedores.

Baixar Mono MRE Baixar Mono MDK

MRE é o Mono Runtime Environment, MDK é o Mono Development Kit.

Use o MDK se você está desenvolvendo aplicações que rodam no .NET/Mono. Ele inclui o MRE e ferramentas adicionais, bibliotecas, perfis .NET PCL (Portable Class Library), etc. que são necessários para desenvolvimento.
