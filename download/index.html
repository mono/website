O Mono para o Linux está disponível para várias distribuições

Por favor procure informações no guia de instalação por mais informações de como instalar e configurar o ambiente do Mono.

Pacotes Xamarin

Estes pacotes são construídos, testados e distribuidos pela Xamarin. Use estes se você quiser usar a versão estável oficial do Mono.

  • Debian, Ubuntu, and derivatives
  • CentOS, Fedora, and derivatives
  • openSUSE and SLES

Pacotes das Próprias Distribuições Linux

O Mono é empacotado para diversas distribuições Linux por mantenedores externos. Alguns pacotes não contém a última versão do Mono, isso depende de cada distribuição/mantenedor.

  • Arch Linux
  • CentOS
  • Debian
  • Gentoo
  • openSUSE
  • Ubuntu

Integração Contínua dos Pacotes

Todo comite em nosso repositório automaticamente causa a compilação e construção dos pacotes a serem distribuídos. Use está versão se você quiser uma versão 'fresquinha' do Mono e não estiver preocupado com a estabilidade da mesma.

Aprenda sobre os pacotes da Integração Contínua
