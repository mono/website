Framework .NET de Código Aberto, Multi-Plataforma

O Mono é uma plataforma de software construída para permitir que desenvolvedores possam criar facilmente aplicações multi-plataforma.

Patrocinada pela Xamarin, o Mono é uma implementação de código aberto do Framework .NET Framework da Microsoft baseado nos padrões ECMA para o C# e a Common Language Runtime. O crescimento de uma família de soluções e um entusiasmada comunidade contribuidora têm ajudado a posicionar o Mono com a principal escolha para o desenvolvimento de aplicações multi-plataforma.

Grande novidade: A Microsoft abriu o código da maior parte do .NET, e isso é imensamente benéfico para o projeto Mono!
